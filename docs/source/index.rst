PHP Riak Client
===============

Installation
------------

Add the client to your project using Composer_::

    $ composer require "fbs/riak-client"

Run ``sphinx-quickstart`` in there::

    $ cd docs
    $ sphinx-quickstart

.. _Riak: http://docs.basho.com/riak
.. _Composer: https://getcomposer.org
